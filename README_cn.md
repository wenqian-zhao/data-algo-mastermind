## 开始

要开始学习，请使用以下命令将此仓库克隆到您的本地计算机：

```bash
git clone https://github.com/wenqian-zhao/data-algo-mastermind.git

```

或者，您可以将代码仓库下载为 ZIP 文件并解压缩。

## 目录结构


其中目录包含有关第X章的笔记，命名为notes.md，"quizzes"目录包含课程编程问题的解决方案。
